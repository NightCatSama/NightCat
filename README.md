前端：Vue 全家桶<br>

## 启动方法

### 服务端

#### 1.先连接 mongodb 数据库

example:
```bash
mongod -f mongodb.conf
```
mangodb.conf
```bash
dbpath = /Users/mac/Mongodb/data
logpath = /Users/mac/Mongodb/mongodb.log
logappend = true
```

#### 2.启动服务器
```bash
npm start
```

### 客户端

#### 1.进入前端项目
```bash
cd ./view
```

#### 2.启动项目
```bash
// 开发环境
npm run dev
// 生产环境
npm run build
```
